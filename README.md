# PPPP-Solidity

该笔记会根据Solidity最新官方文档由浅到深全面复习和总结Solidity，同时也会把智能合约开发相关的知识和目前web3常见的项目进行分析，适合想要深入学习智能合约的同学，同时也会在笔记中说一些实际的智能合约工厂开发岗位遇到的问题，以及一些面试题，欢迎大家指正和交流。
